Importância das tags de links

(VOLTAR)

Tags de Links são muito importantes em um site, elas são responsáveis a usuário ter uma maior navegabilidade e acesso com mais praticidade. elas podem tanto abrir uma nova página, como: < a href="https://www.exemplo.com"target="_blank" > mas também podem abrir em sí própria, removendo o atributo "target", com isso a < tag > vai ter seu valor de origem, que é substituir ela mesma por uma nova página que o usuário clicou. utilizada para links que redirecionam para a própria página, que na maioria dos casos não há necessidade abrir várias páginas do seu site, assim confundindo seu usuário. o ideal, é você apenas abrir novas páginas quando sua página redirecionar o usuário para outro site, assim haverá a chance do usuário voltar para seu site logo depois.

Tags para textos e parágrafos

(VOLTAR)

Existem tags de todos os tipos, e as específicas para criação de textos títulos e parágrafos. as tags mais importantes para criação de um texto são: < h1, > < h2, > < h3, > < h4, > < h5, > < h6 > essas tags são essenciais para a criação de títulos e subtítulos e cabeçalhos, com a h1 sendo a palavra com maior tamanho e prioridade, e a h6 sendo a menor, utilizada para fazer títulos dentro de um título maior, mais conhecido como subtítulos. também existem a tag < p > para criar parágrafos, e tags para personalizar o texto, tanto a cor como formas das letras dando mais ênfase a palavra.

Personalização de texto

(VOLTAR)

Como dito anteriormente^, existem tags que deixam o texto mais bonito, como a cor ou a forma de ser escrito, aqui podemos ressaltar a tag < font > utilizado para mudar a cor, ou até mesmo o tamanho do texto, exemplo: < font color="blue" size="6" face="Arial" > nesse caso, estaria transformando o texto em azul, e aplicando o tamanho 6 pt
